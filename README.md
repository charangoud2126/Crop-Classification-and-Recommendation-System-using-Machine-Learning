# 🌾Crop Classification & Recommendation System using Machine Learning

## Project Overview

Agriculture plays a crucial role in the economy, and selecting the right crop based on soil and environmental conditions can significantly improve productivity.

This project builds a Machine Learning based Crop Recommendation System that predicts the most suitable crop to grow based on various soil and climate parameters.

The model analyzes features such as Nitrogen, Phosphorus, Potassium levels, temperature, humidity, soil pH, and rainfall to recommend the optimal crop.

## Objective

The main objective of this project is to:

Analyze agricultural soil and climate data

Train Machine Learning models to classify crops

Recommend the best crop based on environmental conditions

Support farmers in making data-driven agricultural decisions

## Dataset

The dataset used in this project contains agricultural parameters used to recommend crops.

Features in the Dataset
Feature	Description
Nitrogen (N)	Nitrogen content in soil
Phosphorus (P)	Phosphorus content in soil
Potassium (K)	Potassium content in soil
Temperature	Average temperature (°C)
Humidity	Relative humidity (%)
pH	Soil pH value
Rainfall	Rainfall in mm
Label	Recommended crop

Dataset file used:

Crop_recommendation.csv
